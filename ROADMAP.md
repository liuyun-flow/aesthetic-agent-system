# Roadmap

## V1.5.2 — 案例库体验打磨
- 案例库数据导入/导出（JSON）
- 案例搜索/筛选增强
- 案例对比结果可视化优化
- 案例图片批量上传
- 数据质量检查（完整性/一致性校验）

## V1.6 — 本地部署准备
- Dockerfile / docker-compose
- .env 配置检查脚本
- 数据库备份/恢复脚本
- 前端 static export 评估

## V1.7 — 本地设置页
- 前端设置页面（VISION_PROVIDER 切换、模型选择）
- BYOK 配置（用户自带 API Key，保存到后端本地 `data/config/app_config.json`）
- 设置页只返回脱敏后的 key，不把 API Key 存到浏览器 localStorage

## V1.7.1 — 首次使用向导 / Help
- 首次使用向导（`/setup`）
- 帮助中心（`/help`）
- 工作台配置状态条
- `GET /system/status` 综合状态端点

## V1.7.2 — 迭代方向选择与提示词
- 结构化迭代方向
- 选择某个迭代方向后生成对应提示词
- 生成结果保存到对应 iterate 历史记录
- 历史详情展示所有方向、选中方向和对应提示词

## V1.8 — 语义搜索
- 前置条件：案例库 ≥ 50-100 个
- 向量嵌入（OpenAI embeddings 或本地模型）
- 基于图片描述 + 审美标注的语义搜索
- "找和这个案例类似的" 功能

## V2.0 — 多人化（远期）
- 用户登录/注册
- 多用户隔离
- 云存储（图片/CDN）
- 付费/订阅
- 部署到云服务

## Future Ideas
- 浏览器扩展（右键分析任意网页设计）
- Figma 插件（直接在 Figma 里分析设计稿）
- 训练课程模板（新手→进阶 30 天训练计划）
- 社区案例分享
