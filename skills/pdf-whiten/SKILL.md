# PDF 背景变白

将扫描版 PDF 的彩色/灰色背景替换为纯白色，同时完整保留内容和排版。

## 🌐 网页版

**https://liuyun-flow.github.io/pdf-whiten/**

手机/电脑浏览器打开即用，无需安装。支持微信分享链接，对方在微信内点开即可使用。

## 适用场景

- 扫描的登记表、证书、合同等有底色（蓝底、灰底、黄底等）的 PDF
- 需要打印但背景色会消耗墨水或看起来不干净的 PDF
- 任何以图片方式存储、背景非纯白的 PDF

## 处理原理

该 PDF 的每一页都是一张扫描图片（JPEG）。处理分三步：

1. **边缘采样** — 从页面四边采样获取背景色的中位数
2. **逐通道白化** — 对每个 RGB 通道独立计算像素与背景色的距离，将接近背景色的像素推向白色
3. **残留清理** — 强力清除剩余的浅灰色调，但不影响深色内容（文字/线条/印章）

## 使用方法

### 基本用法

```bash
python3 "<skill-path>/scripts/whiten_pdf.py" "<输入PDF路径>"
```

输出文件会自动在输入文件同目录生成，文件名添加 `_白底` 后缀。

### 指定输出路径

```bash
python3 "<skill-path>/scripts/whiten_pdf.py" "<输入PDF路径>" "<输出PDF路径>"
```

### 调整激进程度

```bash
python3 "<skill-path>/scripts/whiten_pdf.py" "<输入PDF路径>" "<输出PDF路径>" --level high
```

可选等级：`normal`（默认）、`high`（更激进）、`low`（保守）。

## 依赖

需要 Python 3 和以下库：
- `pymupdf` — PDF 处理
- `Pillow` — 图片处理
- `numpy` — 数值计算

安装命令：
```bash
python3 -m pip install pymupdf Pillow numpy
```

## 安全保障

- **零内容丢失**：深色像素（文字、线条）完全不变，仅处理浅色背景区域
- **原文件保留**：默认不修改原始文件
- **逐页自适应**：每页独立采样背景色，适应扫描光照不均的情况

## 注意事项

- 仅适用于**以图片形式存储**的扫描版 PDF（每页是 JPEG 图片）。不适用于矢量文字 PDF
- 如果 PDF 包含印章或有色内容，这些会被保留（它们与背景色相差较大）
- 输出文件会比原文件大（JPEG 重新编码，质量 92%）
